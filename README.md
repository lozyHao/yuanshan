# 远山·yuanshan

> 一款快捷方便的照片添加边框/水印的可视化预览工具平台，能实现自定义样式、文案和图片。

## 功能简介

1. 自定义边框、背景、图片、文字、水印样式。
2. 自定义效果实时预览、报告生成导出、照片参数拷贝。
3. 自定义参数和模板。
4. 图片大小无限制，画质保留无压缩(可执行设置压缩百分比)。
5. 平台支持暗黑模式、恢复默认状态、文字转图片和画板等工具。
6. 项目开源任意玩。
